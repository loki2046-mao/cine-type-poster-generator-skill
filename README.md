# CineType Poster Generator Skill

**电影字报生成器 / 质感文字海报生成器 / 万物电影海报**

把任意主题转化为真实摄影质感、电影海报感、强字体排版设计感的视觉图。

适用于：城市、地点、人物、物体、事件、节日、抽象概念、情绪、生活场景、影视 IP、小说 IP、品牌气质、一句话主题。

## 使用方式

### 在 Cola 里使用

直接告诉 Cola 一个主题，skill 会自动分析并生成图像提示词，在 Cola 环境中可直接调用 listenhub 生图。

### 在其他 AI 环境里使用

把 `SKILL.md` 内容交给你的 AI agent，它会按框架生成提示词，你可以把提示词复制到任何支持的生图工具（GPT Image 2、Nano Banana Pro、Midjourney、DALL-E、Flux 等）。

## 核心能力

- **生活真实感翻译**：不是把关键词拆成物体，而是翻译成真实生活里会出现的视觉证据
- **文字分层系统**：ZONE 1–4 分区描述，驱动多层文字排版出图质量
- **6 种排版系统**（大字压图 / 竖排切割 / 叠压海报 / 极简电影卡 / 复古旅行 / 强概念海报）
- **6 种色彩系统**（暖胶片 / 冷调纪实 / 黑白图形 / 高对比社交 / 低饱和复古 / 时尚黑白红）
- **IP & 品牌安全规则**：既有影视 IP 默认不生成人脸，品牌主题不伪造 logo
- **字体版权规则**：完整的开源字体方向列表
