# CineType Poster Generator

**核心定位**：输入任意主题 → 判断类型 → 翻译成真实可拍摄场景 → 匹配照片质感 → 设计文字系统 → 生成具有高级设计感的电影字报。

目标效果：电影海报、旅行明信片、杂志封面、唱片封面、复古胶片卡片、高级产品视觉、概念海报、社交媒体视觉卡。  
不要做出：普通 PPT 封面、廉价旅游海报、电商详情页、AI 插画图、字面元素拼贴、伪官方影视海报、模板化宣传单。

---

## 一、核心审美目标（四条必须同时满足）

1. **真实照片质感** — 主图像真实摄影，不像插画、3D、或 AI 合成感重的图
2. **文字是画面主体** — 文字参与构图，压住画面、切割空间、形成视觉张力，不是后贴的说明
3. **电影感和文艺感** — 像被拍下来的一个瞬间，有空气感、时间感、场景感和情绪
4. **明确设计概念** — 有一个清楚的视觉锤，不是堆道具

## 二、主题类型分类

### City / Place 城市与地点
适合风格：旅行明信片、城市电影剧照、复古旅游卡片、街头纪实摄影、黄昏城市风景、雨后街景、黑白胶片街景。  
画面建议：街道、建筑、天际线、窗口、路牌、旧墙面、车站、河流、黄昏灯光、雨后路面。  
标题排版：中文地名大字压画面，外文地名小字穿插，可竖排、横排、错位、叠压，城市名必须清楚可读。

### Object / Product 物体与产品
物体放真实环境，不要白底电商图。可用窗边、桌面、暗光、浅景深、生活痕迹。主体清楚，不被文字完全遮挡。  
注意：有真实品牌时仅表达气质，不伪造 logo，不指定品牌字体，不做官方发布感。

### Person / Character 人物与角色
默认不生成真实人物正脸。用背影、侧影、局部、远景、空间气质表达。环境表达人物状态。

### Event / Festival 事件与节日
用象征物表达事件气氛，不要太直白，避免 PPT 感，用真实场景、光线、局部细节建立气氛。

### Abstract Concept / Emotion 抽象概念与情绪
用场景隐喻，不要直接画概念。情绪优先，解释其次。中文标题要像一句电影片名。字体可更压抑、更留白、更克制。

### Existing IP / Movie / TV / Novel 既有 IP
**默认不复刻人物正脸**，尤其不生成"像某演员但不准确"的伪剧照。  
优先用：代表性场景、关键道具、服装局部、空间气质、色彩系统、角色权力关系、作品主题词、背影或剪影。  
目标是**概念视觉海报**，不是伪造官方海报。

## 三、生活真实感规则（核心中的核心）

**不要把关键词机械拆成可见物体，要翻译成真实生活里会出现的视觉证据。**

### 冰西瓜
- **视觉证据**：刚从冰箱里拿出来的西瓜；西瓜切面有细密冷凝水珠；瓜皮边缘有水汽；盘子外侧有淡淡水雾；桌面留下浅浅冷凝水印；瓜瓤颜色清透鲜亮；夏日午后阳光；半开的窗帘；老式风扇或夏日生活背景
- **限制**：不要冰块，不要把西瓜放在冰堆里，不要雪花，不要蓝色寒气特效

### 热咖啡
- **视觉证据**：杯口有轻微热气；杯壁温润；桌面有清晨光线；手边有书或电脑
- **限制**：不要火焰，不要夸张热浪特效

### 春天的风
- **视觉证据**：窗帘轻轻鼓起；树影晃动；纸页被吹起一角；衣角轻微摆动
- **限制**：不要画风的符号，不要卡通风线

### 失眠
- **视觉证据**：床头灯没关；手机屏幕微亮；被子有翻动痕迹；窗外夜色；水杯、药盒、书页等生活细节
- **限制**：不要夸张黑眼圈，不要恐怖片风格

### 松弛感
- **视觉证据**：人或空间姿态自然；物品有轻微生活痕迹；光线舒服；构图不刻意
- **限制**：不要直接堆躺平、沙发、咖啡等模板元素

### 温暖的家
- **视觉证据**：暖黄色台灯；沙发和毯子；热茶或马克杯；翻开的书；窗外夜色或雨声感；门口一双拖鞋；餐桌上留着的饭菜；厨房里一点蒸汽；墙上普通照片框但不出现清晰人脸；植物、台历、钥匙、小纸条、抱枕等生活痕迹
- **限制**：不要默认加宠物，不要默认加人脸，不要靠模糊猫狗表达温馨

### 冬天火锅
- **视觉证据**：铜锅或鸳鸯锅热气升腾；窗外灰蓝色冬夜或雪后街景；室内暖光；桌上食材自然摆放；玻璃窗有雾气；筷子、碗、蘸料、热茶
- **限制**：不要夸张雪花贴纸，不要火焰特效，不要过度摆拍，不要廉价餐饮广告

## 四、高风险细节规避规则

**除非用户明确要求，否则默认不要添加：人脸、宠物脸、婴儿、手部特写、多人合影、复杂文字招牌、品牌 logo、名人面孔。**

如果必须出现生命体（猫、狗、人物等）：
1. 脸必须清晰、自然、可辨认
2. 不放暗部、边缘、遮挡区域
3. 不要半张脸、糊脸、畸形五官
4. 不确定能否生成清楚时，改成背影、爪子、尾巴，或直接取消
5. 不要用模糊宠物表达温馨感
6. 宠物如果出现，必须是明确的猫或狗，不能生成不明动物

---

## 五、强设计概念规则

当设计感不足时，**不要增加元素，要减少元素，提炼一个强视觉概念。**

设计感来自选择，不来自堆料。

优先使用：一个核心物件 + 一个强构图 + 一个明确色彩冲突 + 一个视觉隐喻 + 大面积留白或大面积黑场 + 更大胆的文字压图/裁切/错位/竖排/遮挡。

### 强概念示例

**穿普拉达的女王2**
- **空王座版**：空的黑色主编椅 + 椅背黑大衣 + 桌面一支红口红 → 女王不在场，气场压住全场
- **红线审稿版**：白色时尚稿件铺满画面 + 一条暗红色审稿线像刀一样切过版面 + 黑色墨镜 → 权力不是坐在那里，是划掉一切
- **高跟鞋压稿版**：一只黑色高跟鞋踩在稿件上 + 稿件边缘有红色批注 + 不露人脸 → 规则被踩在脚下

**温暖的家**
- **暖灯孤岛版**：黑夜窗边，一盏灯照亮沙发和书
- **回家痕迹版**：门口拖鞋、钥匙、热汤蒸汽，不出现人脸
- **餐桌余温版**：桌上一碗热汤、两副碗筷、一张便签

**冰西瓜**
- **冷凝水版**：西瓜切面水珠和桌面冷凝水印是唯一核心视觉
- **夏日风扇版**：西瓜清透切面 + 远处模糊风扇 + 阳光

## 六、排版系统（Layout A–F）

### Layout A: Big Block Title 大字压图

适合：城市、地点、产品、节日、强主题。  
特点：中文标题极大，占画面 30% 到 55%，字体粗重，可以遮挡建筑、天空或背景，外文小字贴在中文标题附近。

```text
bold oversized Chinese title typography, occupying nearly half of the image, integrated into the photographic composition, open-source geometric sans-serif style
```

### Layout B: Vertical Split Title 竖排切割

适合：东方城市、人物、武侠、文艺主题、黑白照片。  
特点：中文竖排，文字贴边或穿插主体旁边，画面更安静、更高级，适合黑白、灰蓝、米色、低饱和。

```text
vertical Chinese title layout, editorial poster composition, elegant open-source serif or sans-serif style, integrated with the subject
```

### Layout C: Overlapping Poster Type 叠压海报字

适合：复杂城市、旅行、热闹场景、复古电影感。  
特点：大字与背景强叠压，文字有局部遮挡、裁切，标题可以超出安全区，有更强设计张力。

```text
overlapping typographic composition, large Chinese characters partially cropped by the frame, layered over a realistic cinematic photograph
```

### Layout D: Minimal Film Card 极简电影卡

适合：抽象概念、情绪、产品、安静主题。  
特点：主图留白更多，标题较克制，微型文字系统更重要，像电影截图、展览海报、独立杂志页面。

```text
minimal cinematic photo card, restrained Chinese typography, small metadata text, quiet negative space, premium editorial layout
```

### Layout E: Retro Travel Label 复古旅行标签

适合：城市、地点、旅行、景点。  
特点：像复古旅行贴纸或明信片，圆角卡片，胶片边框，日期、坐标、route、collection 信息，中文标题醒目。

```text
retro travel label design, rounded postcard frame, realistic location photography, bold Chinese place name, small multilingual travel metadata
```

### Layout F: Strong Concept Poster 强概念海报

适合：用户要求更有 feel、更高级、更设计、更狠。  
特点：一个核心视觉锤，大面积留白或黑场，极少道具，字体和图像形成强冲突，可使用裁切、断裂、红线、遮挡、错位。

```text
strong conceptual poster design, one dominant visual metaphor, minimal objects, bold Chinese typography with cropping and tension, high-end editorial art direction
```

## 八、文字系统规则

### 中文主标题
不一定照抄完整输入，提炼成 2–6 字电影片名式标题：

| 输入 | 推荐标题示例 |
|------|-------------|
| 夏天的冰西瓜 | 夏日 / 一口凉 / 夏日凉意 |
| 冬天里的火锅 | 冬日暖食 / 热气升腾 / 冬夜火锅 |
| 温暖的家的感觉 | 温暖的家 / 有家有光 / 回到灯下 |
| 加班到凌晨三点 | 凌晨三点 |
| AI 让我更暴躁 | 更暴躁了 |
| 失眠 | 睡不着 |

城市/地点：直接用城市名，可加诗意但不影响识别度。

### 外文辅助标题
1–5 个词，增强电影感。例：`summer archive`、`city memory`、`night shift`、`cold summer bite`、`warm home`、`the devil returns`、`winter warmth`、`little red panda`、`urban postcard`。

### 微型信息文字
只做设计系统，不必有强信息量。可自动生成：`2026`、`VOL.01`、`ARCHIVE`、`MEMORY SERIES`、`CINETYPE COLLECTION`、`LAT / LNG`、`ROUTE 07`、`STILL FRAME`、`DAYLIGHT RECORD`、`URBAN NOTES`、`WINTER ARCHIVE / 2026`、`SUMMER ARCHIVE / JUNE 2026 / STILL LIFE`、`HOME MEMORY COLLECTION`、`FASHION OFFICE / NEW YORK / VOL.02`。  
要求：小、克制，不抢主标题，不出现假 logo，不出现真实品牌标识。

## 十、文字分层系统（ZONE）

当画面需要多层文字排版时，用以下 ZONE 格式描述，让图像生成模型明确理解每一层文字的位置、大小、颜色和角色：

```text
TYPOGRAPHY — MULTIPLE TEXT ZONES:
ZONE 1 — [位置，如左上角 / 右上角 / 顶部居中]: [内容描述、字号级别（大/中/小）、颜色、对齐方式]
ZONE 2 — [位置，如左侧纵列 / 右侧边缘 / 中部穿插]: [内容描述、字号级别、颜色、排版方向（竖排/横排）]
ZONE 3 — [主标题区域]: [中文主标题内容、字号比例（如占画面高度40%）、颜色、排版方向、压图/裁切关系]
ZONE 4 — [底部信息区]: [内容描述、字号极小、颜色淡、一行或多行微型信息]
```

**使用原则：**
- ZONE 3 是主角，必须清楚可读，可压图、可裁切，但不能读不出来
- ZONE 1/2 是辅助层，字号比 ZONE 3 小 3–5 倍，颜色可对比也可融入背景
- ZONE 4 是设计系统层，极小，仅作版式支撑，不抢注意力
- 不是每张图都需要四个 ZONE，根据 Layout 选择 2–4 个

### 12.2 夏天的冰西瓜（完整提示词）

```text
生成一张高级电影感图文海报卡片，主题是：夏天刚从冰箱里拿出来的一盘西瓜。

画面主体：
真实摄影质感的夏日下午室内桌面，一盘切好的西瓜放在老式搪瓷盘或白瓷盘里，西瓜没有冰块包围，但必须通过细节让人感觉它很冰：西瓜切面上有细密冷凝水珠，瓜皮边缘有水汽，盘子外侧有淡淡水雾，桌面留下浅浅的冷凝水印，瓜瓤颜色清透鲜亮。旁边可以有一把切瓜刀、一块浅色桌布、半开的窗帘和阳光斑驳的影子，像夏天午后刚从冰箱里拿出来准备吃的一盘西瓜。
```

## 十四、质量检查清单

## 十五、执行流程

1. 读取用户主题
2. 判断主题类型（六类之一）
3. 判断是否包含高风险元素（人脸 / 宠物 / 品牌 / IP）
4. 生活感主题 → 执行生活真实感翻译；既有 IP / 真人 → 执行安全规则
5. 选定：核心视觉概念 + 排版系统（A–F）+ 色彩系统
6. 设计文字分层（ZONE 1–4），生成中文主标题（提炼，不照抄）+ 外文辅助 + 微型信息文字
7. 填入提示词模板，附加 Negative Prompt
8. 对照质量检查清单逐项确认
9. 如果用户要求直接出图，调用生图工具（`listenhub generate_image`，model: `gemini-3.1-flash-image-preview`，默认比例 3:2，默认尺寸 2K）
10. 用户反馈不满意时，先诊断再调整，不只道歉

### 多版本策略
用户要多个版本时：先列出 3–5 个方向，从最强的一版开始，**一张一张生成**，不拼宫格合集。用户明确说"一张一张来"时，严格一次只生成一张。

### 交互改进
- "没有 feel" → 减少元素，提炼强视觉概念，切换到 Layout F
- "宠物脸太糊" → 删掉宠物，改用生活痕迹表达
- "这是西瓜加冰块" → 重新执行生活真实感翻译
- "像某演员但不准确" → 改为空间/道具/背影概念海报
- "一个版本一个版本来" → 严格一次只生成一张

## 十六、IP / 品牌安全规则

- **品牌**：可表达气质和产品类别，不伪造官方 logo，不使用品牌专属字体，不做官方发布物料感
- **影视 IP**：不伪造官方剧照，不生成像真实演员但不准确的脸，目标是概念海报
- **真实人物**：尽量改为概念化表达、背影、剪影、道具、空间；如必须生成，避免声称真实准确

IP 方向示例：
- 穿普拉达的女王2：纽约高层时尚办公室、黑色主编椅、黑大衣、红色口红、审稿红线、墨镜、白黑红配色
- 倚天屠龙记03版：如未明确要求复刻演员脸，优先用武侠场景、刀剑、衣袍、江湖山水、古装背影
- 卡萨布兰卡：黑白电影感、机场、酒馆、风衣、烟雾、光影、钢琴、旧电影字幕感

## 十八、Skill 的设计哲学

这个 Skill 是一个**视觉判断系统**，不只是提示词模板。

> 真实生活比字面拼贴高级  
> 文字参与构图，不是贴在照片上  
> 设计感来自选择，不来自堆料  
> 既有 IP 不一定要放人脸  
> 温暖感不一定靠宠物  
> 高级感需要克制  
> 用户说的 feel，往往来自一个强视觉概念
